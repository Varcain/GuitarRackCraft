# FEX-Emu patches for the FEX pivot

Each `NNN-name.patch` here is a patch we wrote against upstream FEX-Emu
(github.com/FEX-Emu/FEX, pinned to a specific commit) to make it
cross-compile and run on Android Bionic arm64. Apply via
`scripts/apply-fex-patches.sh`.

We may *read* third-party diffs for reference, but we do not vendor
patches verbatim.
